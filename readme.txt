1. npm start で起動
2. https://127.0.0.1:8080/ にアクセス
3. ボタン押してEXボイス再生したり、マイクに向けて喋ったり、テキスト欄に文字入力して送信したりする

その他
・きりたんのEXボイスをexVOICE内に配置してください。
・棒読みちゃんとWebsocketプラグインを使っています
　Websocketプラグインはこちらを導入してください
　https://github.com/chocoa/BouyomiChan-WebSocket-Plugin
・certの発行：httpsアクセスを許可したいアドレスを増やす場合は再発行してください
　https://qiita.com/hbjpn/items/925c8012cd93d9165be6
